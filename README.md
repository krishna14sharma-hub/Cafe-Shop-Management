# ☕ Cafe Shop Management System

## 📌 Description
A full-stack web application to manage cafe orders.  
Users can browse menu items, place orders, and data is handled through a C# Web API.

---

## 🚀 Features
- View coffee menu
- Place orders
- Backend API using ASP.NET Core
- Database integration (coming soon)

---

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: C# (ASP.NET Core Web API)
- Database: SQL Server

---

## 📂 Project Structure
- Controllers → API endpoints
- Models → Data structure
- Services → Business logic

---

## ▶️ How to Run
1. Open project in Visual Studio
2. Run the API
3. Use browser or Postman to test endpoints
